## 💸 Were to use Redis?

"All Redis data resides in memory, which enables low latency and hugh throughput data access." (1)
This advantages led to the following Use Cases:

- Caching
- Chat, messageing, and queues
- Gaming leaderboards
- Session storage
- Rich media streaming
- Geospatial
- Maschine learning
- Real-time analytics
